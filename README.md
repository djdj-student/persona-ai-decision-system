# 🧠 Persona AI Decision System

An AI-powered multi-agent decision-making system that simulates diverse personalities, performs risk-weighted voting, and generates transparent, explainable final decisions.

---

## 🚀 Overview

This project introduces a multi-persona AI decision framework where multiple simulated personalities analyze a user’s question from different perspectives and collaboratively produce a final recommendation.

Instead of relying on a single model output, the system:
- 🧠 Simulates multiple decision-making agents
- ⚖️ Applies weighted voting
- ⚠️ Adjusts decisions based on risk awareness
- 🏁 Produces a final aggregated judgment

---

## ✨ Key Features

- 👤 **Multi-Persona Simulation**  
  Each AI agent represents a distinct personality with unique reasoning styles.

- ⚖️ **Weighted Voting System**  
  Decisions are aggregated using dynamic weights.

- ⚠️ **Risk Adjustment Mechanism**  
  High-risk decisions are penalized to improve reliability.

- 📊 **Conflict Detection**  
  Measures disagreement among agents.

- 🏁 **Final Decision Engine**  
  Outputs a clear “Do / Not Do” recommendation.

- 🎨 **Interactive UI (Streamlit)**  
  Clean and modern interface with visualized results.
